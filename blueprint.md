# Lotto Number Generator

## Overview

This project is a web-based lottery number generator. It provides a user-friendly interface to generate random lottery numbers.

## Design and Features

*   **UI:** The application will have a clean and modern design with a single button to generate numbers. The generated numbers will be displayed in a clear and visually appealing way using Web Components.
*   **Functionality:**
    *   Users can click a button to generate a new set of lottery numbers.
    *   The application will generate 6 unique random numbers between 1 and 45.
*   **Technology:** The application is built using HTML, CSS, and JavaScript, following modern web standards. It uses a Web Component to display the lottery balls.
